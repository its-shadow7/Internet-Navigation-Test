**Internet Veteran** is a satirical browser-based game that tests your ability to navigate the hostile environment of the modern internet. From misleading download buttons and guilt-tripping unsubscribe moduls to tech support scams and difficult captchas, this game challenges your digital street smarts.

## ‚ú® Features

* **Dark Pattern Simulation:** Navigate through deceptive UI designs meant to trick you.
* **Scam Awareness:** Identify fake "Virus Alerts" and "You Won an iPhone" pop-ups.
* **Reflex Testing:** A built-in reaction time test (must be under 269ms!).
* **Custom Captcha:** A visual captcha that actually requires you to identify objects.
* **Damage System:** Wrong clicks trigger a screen-shake effect and turn the screen red.
* **Certificate Generation:** Upon completion, generate a personalized "Certified Internet Veteran" certificate that can be downloaded as a PDF.

## üïπÔ∏è Levels Overview

The game consists of several distinct levels found in `InternetVetern.js`:

1.  **Robot Check:** Find the *actual* close button on a fake security popup.
2.  **Real Download:** Identify the correct download link among ads and fake buttons.
3.  **Cookie Consent:** Try to reject cookies when the UI tries to force you to accept.
4.  **Unsubscribe:** Face the emotional manipulation of an unsubscribe screen.
5.  **Virus Alert:** Close a fake "Critical Error" tech support scam window.
6.  **Catch Me:** A button that runs away from your cursor using mouseover events.
7.  **Reflex Check:** Test your reaction speed against the clock.
8.  **Vegetable Captcha:** Select the correct tiles in a custom grid.
9.  **iPhone Scam:** A simulated WhatsApp sharing scam.
